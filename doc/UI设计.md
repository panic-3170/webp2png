## 1. 设计目标

**一句话**：做一个"打开就知道怎么用、用完就想走"的工具站。

### 1.1 核心目标

1. **首屏即工具**：用户 5 秒内找到上传入口。
2. **零干扰**：没有炫技动效、没有渐变光晕、没有浮动客服气泡。
3. **可信赖**：界面克制、留白克制、信息层级清晰。
4. **快**：视觉重量轻 → 加载快、滚动顺、LCP 友好。
5. **可复用**：组件即模板，新工具页只换数据不换结构。

### 1.2 设计原则

| 原则      | 含义                                       | 反例                       |
| ------- | ---------------------------------------- | ------------------------ |
| **克制**  | 一个页面只做一件事；只有一个主 CTA                   | 同时展示"下载""分享""评论""点赞"   |
| **留白**  | 区块间距 ≥ 96px（桌面）/ 64px（移动）             | 内容塞满整屏                   |
| **对比**  | 信息层级靠**尺寸 + 字重**，不靠颜色                    | 用 6 种颜色区分层级              |
| **可逆**  | 所有操作可撤销、可重试，不出现破坏性动作                  | 上传后立刻清空原始文件              |
| **无装饰** | 不用 box-shadow 做大投影、不用毛玻璃、不用渐变        | 卡片浮起 + 投影 + 渐变边框         |
| **一致性** | 同一类元素（按钮、卡片、链接）只用一个样式                 | 4 种样式的按钮                 |

### 1.3 不做什么（明确清单）

- ❌ 动效时长 > 200ms 的过渡
- ❌ 视差滚动 / 滚动触发动画
- ❌ Hero 大背景图
- ❌ 渐变 / 玻璃拟态 / 拟物化
- ❌ 浮动社交分享栏
- ❌ 弹窗（除错误兜底 Toast）
- ❌ 滚动条定制

## 2. 风格参考

> 参考站点：[王哪走都是上坡] 类型的中文极简个人站。

### 2.1 借鉴的要素

| 要素      | 做法                                   |
| ------- | ------------------------------------ |
| 字体      | 系统字体栈（-apple-system / Inter / sans） |
| 颜色      | 近黑白灰 + 单一中性强调色                       |
| 排版      | 大标题 + 灰色正文 + 大量留白                    |
| 边框      | 1px 极浅灰分割                            |
| 容器      | 居中 max-width 1100px                    |
| 卡片      | 白底 + 1px 边框 + 0 阴影                    |
| 数字展示    | 巨大字号 + 小字标签                          |
| 导航      | 顶栏极简，文字链接 + 末尾社交图标                   |
| 回到顶部    | 右下角小圆按钮，无装饰                          |

### 2.2 适配工具站的调整

参考站是"阅读型"站点；我们工具站需要兼顾"操作型"任务，做以下调整：

| 调整点       | 说明                                       |
| --------- | ---------------------------------------- |
| 拖拽区       | 是工具站的"功能按钮"，需要比参考站更**显眼的边界**（dashed border） |
| 主 CTA    | 需要一个**填色主按钮**（参考站只用文字链）                  |
| 错误态       | 需要明确的颜色（红）+ 文案，不靠灰色调区分                  |
| 加载态       | 极简 spinner（一根细线转圈）而非骨架屏                 |
| 工具完成态     | 强调"任务已完成"的反馈（✓ + 文件元信息）                 |

## 3. 设计 Token

> 落地为 Tailwind 配置 + CSS 变量。所有值给出**具体值**，不留模糊。

### 3.1 颜色

#### 浅色模式（Light）

| Token              | HEX         | 用途                          |
| ------------------ | ----------- | --------------------------- |
| `--bg`             | `#fafafa`   | 页面底色                        |
| `--surface`        | `#ffffff`   | 卡片 / 表面                     |
| `--surface-alt`    | `#f4f4f5`   | 浅灰块（广告位背景、代码块）              |
| `--text-primary`   | `#18181b`   | 标题、按钮文字                     |
| `--text-secondary` | `#52525b`   | 正文                           |
| `--text-tertiary`  | `#a1a1aa`   | 辅助说明、占位                     |
| `--border`         | `#e4e4e7`   | 1px 分割                       |
| `--border-strong`  | `#d4d4d8`   | 输入框边框、hover                 |
| `--accent`         | `#18181b`   | 主 CTA（**用纯黑而非彩色**，与极简一致）     |
| `--accent-fg`      | `#fafafa`   | 主 CTA 文字                    |
| `--success`        | `#16a34a`   | ✓ 成功态                       |
| `--warning`        | `#d97706`   | 警告                           |
| `--danger`         | `#dc2626`   | 错误                           |
| `--focus-ring`     | `#2563eb`   | 键盘焦点环（a11y，不依赖品牌色）           |

#### 深色模式（Dark）

| Token              | HEX         |
| ------------------ | ----------- |
| `--bg`             | `#0a0a0a`   |
| `--surface`        | `#111111`   |
| `--surface-alt`    | `#1a1a1a`   |
| `--text-primary`   | `#fafafa`   |
| `--text-secondary` | `#a1a1aa`   |
| `--text-tertiary`  | `#71717a`   |
| `--border`         | `#27272a`   |
| `--border-strong`  | `#3f3f46`   |
| `--accent`         | `#fafafa`   |
| `--accent-fg`      | `#0a0a0a`   |
| `--success`        | `#22c55e`   |
| `--warning`        | `#f59e0b`   |
| `--danger`         | `#ef4444`   |
| `--focus-ring`     | `#60a5fa`   |

#### 色彩使用规则

- **主色即黑/白**：用 `--accent` 作为按钮/链接色，不引入品牌彩。
- **彩色仅用于状态**：success / warning / danger / focus 这四类语义色。
- **彩色不用于装饰**：禁止用渐变、禁止用饱和色做大块背景。
- **对比度**：所有正文 ≥ 4.5:1，大字 ≥ 3:1（WCAG AA）。

### 3.2 字体

| Token           | 值                                       |
| --------------- | --------------------------------------- |
| 字体栈            | `Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif` |
| 等宽             | `ui-monospace, SFMono-Regular, Menlo, monospace` |
| H1              | `clamp(2rem, 4vw, 3rem)` / 700 / -0.02em    |
| H2              | `clamp(1.5rem, 3vw, 2rem)` / 600 / -0.01em    |
| H3              | `1.25rem` / 600                            |
| 正文             | `1rem` / 400 / line-height 1.6            |
| 小字             | `0.875rem` / 400                          |
| 注释             | `0.75rem` / 400 / text-secondary            |
| 数字大屏           | `clamp(2rem, 5vw, 3rem)` / 600             |

> **字重只用 400 / 500 / 600 / 700 四档**，避免字重碎片化。

### 3.3 间距

> 8px 基准。Tailwind 默认值够用，此处列出常用档位。

| Token     | 值       | 用途                |
| --------- | ------- | ----------------- |
| `space-1` | 4px     | 紧贴（图标与文字）          |
| `space-2` | 8px     | 组件内                |
| `space-3` | 12px    | 行内                 |
| `space-4` | 16px    | 卡片内 padding（小）     |
| `space-6` | 24px    | 卡片内 padding（中）     |
| `space-8` | 32px    | 区块内                |
| `space-12`| 48px    | 区块间距（移动）          |
| `space-16`| 64px    | 区块间距（桌面）          |
| `space-24`| 96px    | 主区块间距              |
| `space-32`| 128px   | 页脚上方               |

### 3.4 圆角

| Token       | 值         | 用途          |
| ----------- | --------- | ----------- |
| `radius-sm` | 4px       | 标签、小徽章      |
| `radius`    | 6px       | 按钮、输入框      |
| `radius-md` | 8px       | 卡片          |
| `radius-lg` | 12px      | 弹层、模态       |
| `radius-full` | 9999px | 头像、pill 按钮 |

### 3.5 阴影

> 默认**不使用阴影**。仅在需要层级（如 Toast、模态）时使用 1 级阴影。

| Token       | 值                                          | 用途        |
| ----------- | ------------------------------------------ | --------- |
| `shadow-none` | `none`                                   | 默认        |
| `shadow-sm` | `0 1px 2px 0 rgba(0,0,0,0.05)`             | Toast     |
| `shadow-md` | `0 4px 6px -1px rgba(0,0,0,0.1)`          | 模态（极少使用）  |

### 3.6 动效

> 全部走 `prefers-reduced-motion` 媒体查询关闭。

| Token            | 值                                  | 用途       |
| ---------------- | ---------------------------------- | -------- |
| `duration-fast`  | 100ms                              | hover 反馈 |
| `duration`       | 150ms                              | 默认过渡     |
| `ease`           | `cubic-bezier(0.4, 0, 0.2, 1)`     | 默认缓动     |
| 允许的动效         | `opacity`, `background-color`, `border-color`, `color`, `transform(translate)` |  |
| 禁止的动效         | `scale`, `rotate > 5deg`, 弹性曲线, 路径动画 |  |

## 4. 布局基础

### 4.1 容器

| 项          | 值                                    |
| ---------- | ------------------------------------ |
| `max-width` | `1100px`                             |
| 水平内边距（移动）  | `px-4`（16px）                         |
| 水平内边距（桌面）  | `px-6`（24px）                         |
| 内容主区宽度     | 单列 720px / 网格 1fr×3                |

### 4.2 断点

| 名称     | 区间            | 备注           |
| ------ | ------------- | ------------ |
| `sm`   | ≥ 640px       | 手机横屏         |
| `md`   | ≥ 768px       | 平板            |
| `lg`   | ≥ 1024px      | 桌面（出现侧边栏广告位） |
| `xl`   | ≥ 1280px      | 大屏            |

### 4.3 栅格

- 工具卡：移动 1 列 / 平板 2 列 / 桌面 3 列
- 统计数字：移动 2 列 / 桌面 4 列
- 预览对比：移动 1 列（堆叠）/ 桌面 2 列

## 5. 组件规范

> 按"用到的组件"列举，每个给出变体、状态、间距。

### 5.1 按钮（Button）

**变体**：

| 变体           | 用途         | 样式                                       |
| ------------ | ---------- | ---------------------------------------- |
| `primary`    | 主 CTA（下载）  | `bg-accent` 实心，文字 `accent-fg`            |
| `secondary`  | 次操作（重选）   | `bg-surface` + 1px 边框 + `text-primary`   |
| `ghost`      | 文字链（导航）    | 无边框，文字色，hover 加 `--border` 下划线          |
| `link`       | 行内链接       | 文字色 + 下划线，hover 加深                       |

**尺寸**：

| 尺寸    | 高    | padding   | 字号     |
| ----- | ---- | --------- | ------ |
| `sm`  | 32px | `0 12px`  | 14px   |
| `md`  | 40px | `0 16px`  | 14px   |
| `lg`  | 48px | `0 20px`  | 16px   |

**状态**：
- hover：`opacity-90` 或 `--border-strong`
- active：`opacity-80`
- focus：2px `--focus-ring` 外环，2px offset
- disabled：`opacity-50`，`cursor-not-allowed`

### 5.2 输入框（Input）

- 高度 40px
- 边框 `1px --border`，focus 时变 `--border-strong` + 焦点环
- 圆角 `radius`（6px）
- 占位文字 `--text-tertiary`

### 5.3 卡片（Card）

- `bg-surface`
- 1px `--border`
- 圆角 `radius-md`（8px）
- padding `space-6`（24px）
- 无阴影
- hover（仅在可点击时）：`--border-strong`

### 5.4 工具卡（ToolCard）

> 用于首页工具导航 + "相关工具"区块。

```
┌──────────────────────────┐
│  ▢  图标（24px，--text-secondary）  │
│                          │
│  工具名（H3，--text-primary）       │
│  一句话说明（14px，--text-secondary）│
└──────────────────────────┘
```

- padding `space-6`
- hover：边框变 `border-strong`，`cursor-pointer`
- 整卡可点击
- 移动端：垂直堆叠；桌面：3 列网格

### 5.5 拖拽区（Dropzone）

> 工具页核心组件。

**默认态**：

```
┌──────────────────────────────────────────┐
│                                          │
│                                          │
│            ↑  Drop WebP here            │
│             or click to select           │
│                                          │
│           Max 20MB · No upload           │
│                                          │
└──────────────────────────────────────────┘
```

- 边框 `2px dashed --border-strong`
- 圆角 `radius-md`（8px）
- 高度 ≥ 280px（移动）/ 320px（桌面）
- 背景 `--surface`
- 内边距 `space-12`
- 文字居中
- 整块可点击

**拖拽悬停态（drag-over）**：

- 边框变 `--accent`
- 背景 `--surface-alt`
- **不**做缩放、不做动效

**加载态（processing）**：

- 显示一个细线 spinner + "Converting..." 文案
- 禁用整块交互

**完成态**：

- 整个 Dropzone 隐藏，被 PreviewCompare 替换

### 5.6 预览对比（PreviewCompare）

```
┌────────────────┬────────────────┐
│                │                │
│   [原图缩略]      │   [PNG 缩略]     │
│                │                │
│  Original      │  PNG           │
│  1024×768      │  1024×768      │
│  245 KB        │  312 KB        │
└────────────────┴────────────────┘
```

- 移动：1 列堆叠（顺序：原图、PNG）
- 桌面：2 列等宽
- 图片区域：高度 240px，`object-contain`，浅灰背景（透明 PNG 可显示棋盘格）
- 棋盘格：8×8 棋盘，颜色 `--surface` / `--surface-alt`
- 元信息：图片下方，14px，左对齐
- 区块间距 16px

**变体**：
- 单图：仅显示 PNG（替代下载按钮场景）

### 5.7 动作栏（Actions）

- 与 PreviewCompare 间距 `space-6`
- 桌面：左对齐，下载按钮 primary + 重选按钮 secondary，间距 12px
- 移动：垂直堆叠，下载按钮在上，主按钮宽度 100%

### 5.8 标签（Badge / Tag）

- 高度 20px
- padding `0 8px`
- 字号 12px
- 圆角 `radius-sm`（4px）
- 用途：状态标识（如 "Free"、"New"、"Beta"）

### 5.9 提示（Toast / Inline Error）

- 顶部贴边或行内显示
- 边框 1px 语义色
- 左侧 4px 竖条（语义色）
- 文字 `--text-primary`
- 4 秒自动消失；可手动关闭
- 不堆叠（同时只 1 个）

### 5.10 进度条（Progress）

- 高度 4px
- 背景 `--surface-alt`
- 填充 `--accent`
- 圆角 `radius-full`
- 仅在批量转换（v1.1）使用

### 5.11 导航（Header / Footer）

#### Header

```
┌─────────────────────────────────────────────────────┐
│  ToolKit    Home  Tools  About         ⌘ X ✉     │  ← 64px 高
└─────────────────────────────────────────────────────┘
```

- 高度 64px
- 底边 1px `--border`
- 桌面：左 Logo + 中间导航（文字链） + 右侧社交图标（24px）
- 移动：左 Logo + 右侧汉堡按钮（点开抽屉）
- **不** sticky（避免遮挡内容）；**不**透明背景

#### Footer

```
┌─────────────────────────────────────────────────────┐
│  © 2026 ToolKit · Privacy · About · Contact        │
│  Files never leave your device.                    │
└─────────────────────────────────────────────────────┘
```

- 单行（移动 2 行）
- 顶边 1px `--border`
- 字号 13px
- `--text-tertiary`

### 5.12 广告位（AdSlot）

- 容器：`bg-surface-alt` + 1px 虚线 `--border` + 圆角 `radius-md`
- 顶部 4px 小字："Sponsored"（`text-tertiary`，左对齐）
- 内边距 8px
- 居中广告内容
- 桌面：728x90 / 336x280 / 300x600
- 移动：320x100（响应式）

### 5.13 FAQ

```
Q: Is this tool really free?
─────────────────────────────────────
A: Yes. No signup, no payment...
```

- Q：16px / 600 / `text-primary`
- A：14px / 400 / `text-secondary` / line-height 1.7
- 每条之间 1px 分割线
- 整块**默认展开**，**不**做折叠交互（极简，省 JS）

### 5.14 回到顶部（BackToTop）

- 48×48px 圆按钮
- `bg-accent`，`accent-fg` 图标
- 右下角固定：`bottom-6 right-6`
- 仅滚动 > 600px 时显示
- 点击平滑滚动到顶（150ms）
- 仅桌面端显示

## 8. 暗色模式

### 8.1 实现

- `prefers-color-scheme` 自动切换
- 不做手动开关（v1 简化）
- 用 CSS 变量驱动，Tailwind `darkMode: 'media'`

### 8.2 特殊处理

| 元素          | 浅色          | 暗色                            |
| ----------- | ----------- | ----------------------------- |
| 页面底色        | `#fafafa`   | `#0a0a0a`                     |
| 卡片          | `#ffffff`   | `#111111`                     |
| 拖拽区背景       | `#ffffff`   | `#111111`                     |
| 棋盘格（透明 PNG） | 白 / 浅灰      | `#111` / `#1a1a1a`            |
| 主 CTA       | 黑底白字        | 白底黑字（避免大块高亮色）                |
| 边框          | `#e4e4e7`   | `#27272a`                     |
| 链接          | `#18181b`   | `#fafafa`                     |
| 错误 Toast    | 红边 + 白底     | 红边 + `#111` 底，文字白            |

### 8.3 图片策略

- 浅色：棋盘格用 `#fff` / `#f4f4f5`
- 暗色：棋盘格用 `#111` / `#1a1a1a`
- 不做图片滤镜反转

## 9. 微交互规范

> 总原则：动效是"结果"不是"过程"——出现即可，不要慢动作。

### 9.1 允许的交互

| 元素       | 触发    | 反馈                              | 时长   |
| -------- | ----- | ------------------------------- | ---- |
| 按钮       | hover | 边框/底色变化（用 token 里的颜色）            | 100ms |
| 按钮       | focus | 2px 蓝色焦点环                       | 即时   |
| 工具卡      | hover | 边框变 `border-strong`             | 100ms |
| 拖拽区      | drag-over | 边框 + 背景变化（不加 transform）     | 100ms |
| 链接       | hover | 下划线变 `text-primary`             | 100ms |
| Toast    | 出现/消失 | `opacity 0→1 / 1→0`（不 slide） | 150ms |
| 回到顶部     | 显示   | `opacity 0→1`                  | 100ms |

### 9.2 禁止的交互

- ❌ 按钮 ripple 效果
- ❌ 卡片浮起（translateY）
- ❌ 拖拽区缩放（scale）
- ❌ 加载 spinner 用旋转大圆（用 1 根 2px 细线）
- ❌ 滚动驱动动画
- ❌ 鼠标跟随效果

### 9.3 键盘焦点（必做）

- 所有可交互元素必须有 `:focus-visible` 样式
- 焦点环：`2px solid var(--focus-ring)` + `2px offset`
- 颜色对比度 ≥ 3:1

## 10. 广告位视觉处理

> 目标：让广告"在场但不刺眼"，提升 CTR 同时不破坏极简感。

| 位置        | 容器                            | "Sponsored" 文字 | 备注              |
| --------- | ----------------------------- | -------------- | --------------- |
| 顶部 Header | 容器高度 110px，居中                | 顶部 4px 小字       | 移动 320×100     |
| 完成态下方     | 容器高度 300px，居中                 | 同上             | 336×280，黄金位     |
| 内容中部      | 容器高度 270px，居中                 | 同上             | 300×250         |
| 侧边栏       | 600px 高 sticky                | 顶部             | 仅桌面，固定右侧        |
| 站内相关工具    | 不做"广告"包装，做"Related Tools"区块 | 隐藏             | 视觉上等同内容         |

- 容器背景 `--surface-alt`
- 边框 `1px dashed --border`
- 圆角 `radius-md`
- 内部 padding `space-2`
- 占位阶段（AdSense 未通过）显示一个空灰块 + "Ad placeholder" 文案

## 11. 可访问性（A11y）

### 11.1 必做

- 所有图片有 `alt`（预览图 alt = "Original WebP image" / "Converted PNG image"）
- 拖拽区有 `role="button"`、可 Tab 聚焦、回车触发
- 错误用 `aria-live="polite"` 区域
- 颜色对比度 ≥ 4.5:1（正文）/ 3:1（大字）
- 焦点可见
- 标题层级不跳级（H1 → H2 → H3）

### 11.2 禁用

- 不可用 `tabindex > 0`
- 不可把交互元素放 `<div>`（用 `<button>` / `<a>`）

### 11.3 键盘走查

- Tab：Header → Dropzone → Actions → FAQ 标题 → Footer
- Enter / Space：触发拖拽、按钮
- Esc：关闭 Toast、关闭抽屉
